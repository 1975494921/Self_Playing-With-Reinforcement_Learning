1.Learning_2d.py为训练代码（需要运行的），Parameters.py包含了多种参数，包括寻宝地图的大小，宝藏的位置以及是各个地雷的位置

2.win请使用CMD运行，mac和linux请使用终端，不要用pycharm一类的集成开发程序运行，因为运行调试的窗口不支持清屏命令。

3. 主程序中Load_And_Save_Model 代表是否需要加载和保存模型（文件存储），设置为True代表需要。使用模型文件请在CMD窗口中cd到程序的目录下，否则会加载和保存无效。

4.默认已经训练好了当前参数下的最优解，按步骤三的操作后可以直接得到最优的策略。如果在Parameters.py中改变了地图的大小，宝藏的位置一类的参数请先删掉Model.npl文件 来重新训练，否则会出错。
  如果不使用模型文件将不会出现此问题，可在更改Parameters.py里面的参数后直接使用。

5.Python环境需要安装numpy，不需要其它机器学习库，学习算法本身全部写在代码当中。
