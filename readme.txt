
0.Python环境需要安装numpy，不需要其它机器学习库，学习算法本身全部写在代码当中。

1. win请使用CMD运行，mac和linux请使用终端，不要用pycharm一类的集成开发程序运行，因为运行调试的窗口不支持清屏命令。
    使用前请先在命令行 cd 到程序的主目录，才能调用模型文件，再运行代码。

2. Parameters.py包含了多种参数，如寻宝地图的大小，宝藏的位置，可以自行更改。
    Bomb.py包含了地雷的个数，位置坐标，以及是否启用等设置，可以自行更改。
    如果更改了参数请先删掉模型文件 Model.npy，再重新训练

3. Learning_2d.py：边训练并显示计算机玩游戏的过程，可选择是否保存模型文件。 
    training_no_view.py：只训练不显示计算机玩游戏的过程，一定会保存模型文件。
    final_show.py ：展示计算机训练的最终效果，模型文件必须存在。
    建议使用 training_no_view.py 来训练 和 final_show.py 来展示最终效果，因为地图参数比较复杂，训练并显示跑好几天等于不显示跑几分钟。
    （只训练不显示的训练速度 大概是 训练并显示的 一万倍左右吧，因为打印到屏幕上的过程对于计算机是最费时的）

     默认已经训练好了当前参数下的最优解，可直接运行final_show.py展示计算机玩游戏的效果。

4.想从头自己训练可以删掉Model.npy再运行 training_no_view.py。
   训练默认会尝试加载Model.npy模型文件并继续训练，在没有的情况下才会从头开始训练。
